О компании

ООО «Парексель интернэшнл (РУС)» сосредоточен на разработке новых инновационных методов лечения для улучшения здоровья людей во всем мире. Компания предоставляет набор новаторских услуг по разработке биофармацевтических препаратов, которые помогают клиентам по всему миру преобразовывать научные открытия в новые методы лечения пациентов.

Решение

Была автоматизирована загрузка документов ГТД от брокера.

При проработке интеграции с SAP столкнулись с проблемой уникальности документов. Данные в SAP не соответствовали документам в 1С, так как в них было в некоторых случаях больше функций, а в некоторых меньше, по этой причине были проблемы уникальности документов и соответственно при загрузках затирались данные в документах. Решение было найдено с помощью хэш функций реализованных в 1С, уникальность документов определялась по нескольким реквизитам, в том числе перечень номенклатуры и по ним генерировался хэш и в дальнейшем по нему осуществлялся поиск документов.

По части операций были автоматизированы сверки по SKU c SAP.

При работе над блоком НДС были реализованы обработки по списанию НДС, восстановлению НДС и реализация механизма Tax free.

Также была реализован механизм формирования отчетности PNL с учетом МСФО стандартов международной группы и реализована выгрузка этих данных в SAP.
